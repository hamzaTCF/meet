How We Use Public Calendar Events Information

  • As stated earlier, this application doesn't fetch any personal data, and only fetches publicly available events from the public calendar ID "fullstackwebdev@careerfoundry.com"
  • This application allows the user to do simple queries on the fetched events such as, filtering them based on the selected city or the specified number of events.
  • Finally, the application will display simple charts visualizing the distribution of events per city.
